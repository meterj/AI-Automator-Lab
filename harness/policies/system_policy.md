# System Policy Template

## 목적
이 문서는 프로젝트 전반에 적용되는 상위 정책을 정의한다.

## 핵심 원칙
- 모델 결과는 초안으로 취급한다.
- 외부로 나가는 출력은 검증 후에만 허용한다.
- 파괴적 변경은 승인 전 자동 실행하지 않는다.
- 민감정보는 평문으로 출력하지 않는다.
- 상태는 대화가 아니라 파일에 남긴다.

## 금지 사항
- 근거 없는 사실 단정
- 허가되지 않은 시스템 접근
- 검증 실패 결과의 외부 전달
- 로그 없는 대규모 변경

## 예외 처리
- 긴급 장애 대응 시 임시 예외 허용 가능
- 단, 사후에 run_log / diff_summary / eval_report를 반드시 보강한다.
