# ui_review_checklist.md

## 시각 품질
- [ ] 시각적 위계가 명확하다
- [ ] 핵심 CTA가 눈에 띈다
- [ ] 불필요한 장식이 없다
- [ ] 간격 체계가 일관된다
- [ ] 상태 색상이 일관된다

## 상호작용
- [ ] hover/focus/disabled/loading/error 상태가 있다
- [ ] 사용자가 지금 무슨 상태인지 알 수 있다
- [ ] 오류 메시지가 구체적이다
- [ ] 되돌리기 또는 취소 경로가 있다

## 접근성
- [ ] 명도 대비가 충분하다
- [ ] 키보드만으로 주요 흐름이 가능하다
- [ ] 스크린리더 라벨링이 있다

## 검증 자료
- [ ] 주요 화면 스크린샷 첨부
- [ ] 이전 버전 대비 diff 기록
- [ ] UX 리스크 메모 남김
